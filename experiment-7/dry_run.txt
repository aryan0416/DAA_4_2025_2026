Dry Run Example:
  n = 5 cities  (0, 1, 2, 3, 4)
  src = 0, dst = 4, k = 2

Flights:
  0 → 1  cost 10
  0 → 2  cost 30
  1 → 3  cost 20
  2 → 3  cost 10
  3 → 4  cost 15
  1 → 4  cost 80


Step 1: Initialize
  dist = [0, infinity, infinity, infinity, infinity]
           0     1         2         3         4



Round 1 (i=0) → at most 1 edge used
  temp = [0, infinity, infinity, infinity, infinity]

  0→1 (10):  dist[0]=0         →  temp[1] = min(infinity, 0+10)  = 10  ✓
  0→2 (30):  dist[0]=0         →  temp[2] = min(infinity, 0+30)  = 30  ✓
  1→3 (20):  dist[1]=infinity  →  skip
  2→3 (10):  dist[2]=infinity  →  skip
  3→4 (15):  dist[3]=infinity  →  skip
  1→4 (80):  dist[1]=infinity  →  skip

  dist = [0, 10, 30, infinity, infinity]



Round 2 (i=1) → at most 2 edges used
  temp = [0, 10, 30, infinity, infinity]

  0→1 (10):  dist[0]+10=10   = temp[1]=10        → no change
  0→2 (30):  dist[0]+30=30   = temp[2]=30        → no change
  1→3 (20):  dist[1]=10      →  temp[3] = min(infinity, 10+20)  = 30  ✓
  2→3 (10):  dist[2]=30      →  temp[3] = min(30, 30+10)        = 30  → no change
  3→4 (15):  dist[3]=infinity →  skip
  1→4 (80):  dist[1]=10      →  temp[4] = min(infinity, 10+80)  = 90  ✓

  dist = [0, 10, 30, 30, 90]


Round 3 (i=2) → at most 3 edges used (= k+1 = 2 stops)
  temp = [0, 10, 30, 30, 90]

  0→1 (10):  dist[0]+10=10   = temp[1]=10  → no change
  0→2 (30):  dist[0]+30=30   = temp[2]=30  → no change
  1→3 (20):  dist[1]+20=30   = temp[3]=30  → no change
  2→3 (10):  dist[2]+10=40   > temp[3]=30  → no change
  3→4 (15):  dist[3]=30      →  temp[4] = min(90, 30+15)  = 45  ✓
  1→4 (80):  dist[1]+80=90   > temp[4]=45  → no change

  dist = [0, 10, 30, 30, 45]



Step 3: Result
  dist[4] = 45 ≠ infinity  →  return 45

  Optimal path: 0 → 1 → 3 → 4
  Cost:         10 + 20 + 15 = 45
  Stops:        1, 3  (2 stops = within k=2)  ✓
